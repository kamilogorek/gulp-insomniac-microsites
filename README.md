# gulp-insomniac-microsites
> Easily deploy Insomniac's microsites using S3

## Install

```sh
$ npm install --save-dev gulp-insomniac-microsites
```


## Usage

```js
var gulp = require('gulp');

gulp.task('default', function () {
    return gulp.src('./src')
        .pipe(gulp.dest('dist'));
});
```
